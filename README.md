## 개요
- 간단한 LMS(학습 관리 시스템) 프로젝트
- 코로나로 인해 사이버 공간 학습으로 인한 LMS 활성화

## Use Skills
- Frontend
  - HTML, CSS, JQuery, Thymeleaf
  
- Backend
  - Java (jdk 11), Spring Boot, Spring Security JPA
  - MariaDB, mybatis

## 개발 기간
- 2022.07-04 ~ 2022.07.18
