## 개요
학습 관리 시스템 서비스 프로젝트

## Use Skills
- Spring, Jpa, MariaDB, mybatis
- Java11, Junit5, Gradle
- Thymeleaf, css, html

## Goal
- 관리자가 등록해놓은 강좌를 회원이 구매하고 관리자가 회원과 강좌를 관리하는 것을 구축한다.
